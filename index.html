MINT REST Example

Hello and welcome to the MINT REST demo. This basic JQuery-powered front-end has been designed to showcase how the MINT REST API allows for the creation of simple, streamlined front-ends and user experiences that leverage the unified operator services enabled by the MINT platform.

The following example contains four basic elements:

  1. An administrative "create user" page
  2. A simple user login screen
  3. A rudimentary EPG viewer
  4. A simple video streamer

The basic use cases demonstrated with this appliance are that of an administrator creating a new user, that user logging in, browsing the EPG, and viewing a show.

This sample code has been designed to be as simple as possible, using only a standard REST library to keep things streamlined. The "spaghetti" code was purposely written to ease semantic review. This code SHOULD NOT be deployed to a production environment under any circumstances; it is strictly for education purposes only.
